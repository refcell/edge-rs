# Introduction

Edge is a domain specific language for the Ethereum Virtual Machine (EVM).

It is the brain child of [jtriley](https://github.com/jtriley-eth), to whom
all specifications are attributable.

The [Edge Book] is broken down into the following sections:

* [Specifications](specs/overview.md): An in-depth blueprint to the Edge Language.
* [The Compiler](compiler/overview.md): The inner working of the rust edge compiler implementation.
* [Tooling](tools/overview.md): An overview of Edge Language tooling and other developer utilities.
* [Contributing](contributing/contributing.md): Repository contribution guidelines.
* [Contact](contact/contact.md): Methods of contacting the maintainers.
