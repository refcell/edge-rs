# Introduction

<div className="edge-home-subtitle">
  Language overview, specifications, compiler internals, and tooling for Edge.
</div>

Edge is a domain-specific language for the Ethereum Virtual Machine (EVM):
high-level, strongly statically typed, and designed to make smart contract
development more expressive without giving up control over execution.

The language is the brainchild of
[jtriley](https://github.com/jtriley-eth), to whom the current specifications
are attributable.

Start with the language reference, then drill into the compiler and the rest of
the developer workflow from there.
